# Politique de Confidentialité - KaayLudo

**Dernière mise à jour : 4 mars 2026**

Cette politique de confidentialité décrit comment l'application **KaayLudo** gère les informations des utilisateurs sur iOS et Android. En utilisant KaayLudo, vous acceptez les pratiques décrites ici.

### 1. Collecte des données

KaayLudo ne demande pas d'inscription obligatoire et ne collecte pas de données personnelles nominatives (comme votre adresse ou votre numéro de téléphone) de manière directe.

### 2. Services tiers

L'application utilise les services suivants qui peuvent collecter certaines informations techniques :

**Google Mobile Ads (AdMob)**
- Identifiants publicitaires de l'appareil
- Données utilisées pour afficher des publicités pertinentes
- Politique de Google : [https://policies.google.com/privacy](https://policies.google.com/privacy)

**Firebase (Google)**
- Données de crash et de performance
- Données d'utilisation anonymes
- Politique de Firebase : [https://firebase.google.com/support/privacy](https://firebase.google.com/support/privacy)

**Apple App Store (iOS uniquement)**
- Identifiant publicitaire Apple (IDFA)
- Données de diagnostic
- Politique d'Apple : [https://www.apple.com/legal/privacy](https://www.apple.com/legal/privacy)

### 3. Utilisation des données

Les données techniques collectées servent uniquement à :
- Assurer le bon fonctionnement du jeu
- Afficher des publicités (application gratuite)
- Analyser les performances techniques pour éviter les bugs
- Suivre vos scores et votre progression

### 4. Publicités

KaayLudo est gratuit et financé par la publicité via Google AdMob. Les publicités affichées peuvent utiliser votre identifiant publicitaire pour vous proposer des annonces pertinentes. Vous pouvez limiter le suivi publicitaire dans les réglages de votre appareil :
- **iOS** : Réglages → Confidentialité → Publicité → Désactiver le suivi
- **Android** : Paramètres → Google → Annonces → Désactiver la personnalisation

### 5. Sécurité

Nous nous engageons à protéger vos informations techniques. Cependant, aucune méthode de transmission sur Internet n'est sûre à 100%.

### 6. Enfants

KaayLudo est un jeu familial. Nous ne collectons pas sciemment d'informations identifiables auprès d'enfants de moins de 13 ans. Si vous êtes un parent et que vous pensez que votre enfant nous a fourni des données, contactez-nous.

### 7. Modifications

Nous pouvons mettre à jour cette politique à tout moment. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.
